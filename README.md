# 🎮 Video Game Sales & Engagement Analysis

An end-to-end **data analytics and visualization platform** that transforms raw video game sales data into **actionable insights** using Python, SQL, and Streamlit.

This project demonstrates real-world skills in **data engineering, analytics, and dashboard development**, making it ideal for **Data Analyst / Data Scientist roles**.

---

## 🎯 Business Problem

Gaming companies generate large volumes of sales data but often lack tools to:
- Analyze regional performance  
- Identify top-performing games and genres  
- Ensure data quality and consistency  
- Extract meaningful business insights  

👉 This project solves these challenges through a centralized analytics system.

---

## 💡 Solution Overview

- Built a **data pipeline** for cleaning and preprocessing  
- Designed a **normalized SQLite database**  
- Developed a **multi-page Streamlit dashboard**  
- Integrated **AI-powered insights generation**  
- Implemented **data validation and reporting system**

    ## 🛠️ Code Remarks

- 🔐 **Auth**: `utils/auth.py` initializes `config/users.yaml` on first run with **bcrypt-hashed credentials**.  
  ⚠️ Ensure default usernames and passwords are updated before production deployment.

- 🗄️ **DB Build**: `utils/data_loader.py` executes the schema, performs data cleaning and transformation,  
  deduplicates dimension tables (platforms, genres, publishers), and inserts `games` and `sales`  
  tables with proper **foreign key relationships**.  
  A `(name, platform)` mapping is used to link `sales.game_id → games.game_id`.

- 🧪 **Validation**: `utils/validation.py` generates a **data quality score (0–100)** along with  
  column-level reports for missing values, data types, and duplicates.  
  Results are displayed on the *Data Quality* dashboard.

- 🤖 **Insights**: `utils/insights.py` provides deterministic statistical summaries and supports  
  optional **LLM-based narrative insights**.  
  Designed with graceful fallback in case of API unavailability.
⚠️ **Note:** Update default credentials before production deployment.
